## :anchor: Bắt đầu làm việc

1. Clone repository

```console
git clone https://github.com/dinhkarate/web-dat-san.git
```

2. Tạo branch mới

```console
git branch <branch-name>
```

3. Chuyển sang branch vừa tạo và làm việc trên branch đấy

```console
git checkout <branch-name>
```
